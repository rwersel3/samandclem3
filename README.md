# Sam & Clem - Interactive Children's Storybook

An interactive, animated children's storybook web application featuring the heartwarming tale of Sam the mouse and Clem the cat.

## About

This interactive storybook tells the story of a lonely field mouse named Sam who finds an unexpected friend in Clem the calico cat. The 20-page digital book features:

- 🎨 Beautiful watercolor-style illustrations
- 🎮 Interactive mini-games on each page
- 🎵 Bird orchestra conducting
- 🌸 Animated flower petals and characters
- 📱 Mobile-first responsive design
- 💾 Progress saving (pick up where you left off)
- ♿ Accessibility features

## Local Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm run dev

# Build for production
pnpm run build
```

## Features

### Interactive Elements
- **Page 3:** Help Sam find his snacks (drag & drop mini-game)
- **Page 4:** Meet the garden neighbors (tap to hear animal sounds)
- **Page 7:** Follow Hoppy's path (trace interaction)
- **Page 8:** Build the Cardinals' nest (construction game)
- **Page 13:** Find Whistle Pig's kid (hot/cold treasure hunt)
- **Page 18:** Conduct the bird orchestra (rhythm game)

### Design
- Hand-drawn character animations
- Smooth page transitions with 3D effects
- Progress tracking with flower icons
- Character gallery in credits
